# 日程管理Web应用

一个功能完整的日程管理Web应用，支持日常事项管理、重复事项、多视图展示等功能。不要管launchreadme，直接用start.bat打开，用shutdown.bat关掉。

## 功能特性

### 核心功能
- ✅ 日常日程管理
- ✅ 支持一次性事项和重复事项
- ✅ 实时数据同步
- ✅ 多视图展示（工作台、日历、月历）

### 视图功能
- **工作台**：左侧显示周历安排，右侧显示待办事项列表
- **日历**：显示当日详细信息，包含统计和操作
- **月历**：月视图展示，支持日期导航和快速跳转

### 事项管理
- ✅ 添加/编辑/删除事项
- ✅ 重复事项处理（每天/每周/每月/每年）
- ✅ 事项分类（个人、主业工作、副业工作、社交、恋爱）
- ✅ 完成状态标记
- ✅ 时间管理和未安排事项标识

### 数据管理
- ✅ 本地数据库存储（IndexedDB）
- ✅ 数据导入导出（JSON格式）
- ✅ 数据备份和恢复

### 用户体验
- ✅ 响应式设计，适配不同屏幕
- ✅ 平滑动画和过渡效果
- ✅ 人性化交互设计
- ✅ 实时反馈和提示

## 技术实现

- **前端技术**：HTML5, CSS3, JavaScript (ES6+)
- **数据存储**：IndexedDB（本地数据库）
- **样式设计**：扁平化设计，糖果色系，圆角矩形
- **动画效果**：CSS3动画和过渡
- **字体**：内置"Oz焦糖下午茶"字体

## 使用方法

1. **打开应用**：直接在浏览器中打开 `index.html` 文件
2. **添加事项**：点击导航栏的"添加事项"按钮
3. **切换视图**：使用导航栏的三个视图按钮
4. **管理事项**：点击事项进行编辑或删除
5. **数据备份**：使用导入导出功能进行数据备份

## 注意事项

- 首次使用会自动初始化数据库
- 数据存储在浏览器本地，清除浏览器数据会删除所有事项
- 建议定期使用导出功能进行数据备份
- 重复事项编辑时需要选择编辑范围（仅本次/此后所有）
